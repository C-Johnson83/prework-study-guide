HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Using the marquee tag allows the words to scroll across the screen.

CSS

  • CSS rule: property first then value
  • External CSS style sheets are preffered, but can also be changed in the tag using style=" property:value ;"
  • link rel="stylesheet" href="file path to css" is used to link external CSS
  • Margin is the space outside of the element, Padding is the space inside the element

Git

  • git status: checks what branch we are currently on and if there are files modified
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: adds files to be committed to the repo
  • git commit -m "your comment here ": commits the files with a comment
  • git push: pushes the committed files to the github repo

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • script src="path to JS file" is used to link the JS file.
